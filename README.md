# CV Website

Mobile-friendly CV website based on the original PDF content.

## Local preview
1. Open the folder in VS Code.
2. Open `index.html`.
3. Use Live Server (`Open with Live Server`) or run `start index.html` in PowerShell.

## Publish on GitHub Pages
1. Push this repository to `main`.
2. In GitHub: `Settings -> Pages`.
3. Set source to `Deploy from a branch`.
4. Choose branch `main` and folder `/ (root)`.
5. Save and wait for your public site URL.
